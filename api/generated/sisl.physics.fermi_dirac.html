fermi_dirac

sisl.physics.fermi_dirac(E, kT=0.1, mu=0.0)

Fermi-Dirac distribution function

\[n_F(E,k_BT,\mu) = \frac1{\exp\Big[\frac{E - \mu}{k_BT}\Big] + 1}\]
Parameters:

  • E (ArrayLike) – energy evaluation points

  • kT (float) – temperature broadening

  • mu (float | ArrayLike) – chemical potential

Returns:

the Fermi-Dirac distribution, same length as E

Return type:

numpy.ndarray
