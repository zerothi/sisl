Retrieve a class from the global lookup table via filename and the extension

Parameters:

  • filename (str) ‚Äì

    the file to be quried for a correct file object. This file name may contain {<class-name>} which sets cls in case cls is not set. For instance:

    water.xyz

    will return an xyzSile.

  • cls (class, optional) ‚Äì In case there are several files with similar file-suffixes you may query the exact base-class that should be chosen. If there are several files with similar file-endings this function returns a random one.
