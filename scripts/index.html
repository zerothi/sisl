Command line scripts

sisl implements a set of command-line utitilies that enables easy interaction with all the data files compatible with sisl.

– Geometry manipulation

Process files containing geometries.

Manipulate, by extending/removing atoms and write to a large variety of file formats.

Can be used as a file format conversion tool.

sgeom
– Grid manipulation

Process files containing grid-data.

Manipulation of the grids, by subtracting/reducing grids.

Can be used as a file format conversion tool.

sgrid
